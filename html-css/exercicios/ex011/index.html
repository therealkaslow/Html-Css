  • Lembrando que a ordem das sources, tem que ser da maior imagem para a menor imagem por exemplo: 
  •                     <picture>
                        
                        <source media="(max-width: 1050px)" srcset="assets/foto-m.png" type="image/png">

Reproduzinho Audio

Vamos aprender a reproduzir audios em HTML5

para fazer com que o audio seja reproduzido de fundo, só adicionar a tag audio e colocar o parâmetro controls para aparecer o player de audio e se quiser pode colocar o autoplay para iniciar o audio de fundo

Deixando compatível em todos os navegadores.
  • tag Source

    nem todos os navegadores tem a compatibilidade para rodar o audio, ou seja se você quiser fazer com que o audio toque, os navegadores suportam os formatos OGG, WAV e MP3.

    E para isso você precisa de colocar a tag de audio vazia e depois colocar os respectivos formatos dentro da tag de Source src
