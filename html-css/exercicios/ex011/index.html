Bounce Rate

é quando o usuário automaticamente sai do seu site por questões de desinteresse e o google consegue fazerr um analytics disso, fazendo com que ele vai parar de indicar o site para outras pessoas.

    Como abaixo tem como deixar a imagem responsiva sem o uso do CSS, utilizando a tag de Picture e após criar uma tag de Source definindo a largura máxima.
  • Lembrando que a ordem das sources, tem que ser da maior imagem para a menor imagem por exemplo: 
  •                     <picture>
                        
                        <source media="(max-width: 1050px)" srcset="assets/foto-m.png" type="image/png">
                        <img src="assets/foto-g.png" alt="G" />

                    </picture>
